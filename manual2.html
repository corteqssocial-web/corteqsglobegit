CorteQS Globe / Manual 2

Mevcut pin davranışı API değil, etkileşim akışı kaynaklı.

Son değişikliklerden sonra uygulamadaki beklenen akış şu: cluster tıklanınca harita yakınlaşır, tekil pin tıklanınca ise detay paneli açılır. Tekil pin için yakınlaşma işlemi panel içindeki Buraya yakınlaş butonuyla yapılır.

Sorunun Kaynağı

“Mevcut pine tıklayınca yakınlaşmıyor” gözlemi, şu anki kodun beklenen davranışıyla uyumlu. Bu yüzden ilk bakışta sorun backend API, Supabase ya da geocoding tarafında görünmüyor.

Gerçekte ne oluyor?

  1. DiasporaGlobe.jsx içinde tekil pin tıklaması onPinClick çağırıyor.
  2. App.js içindeki onPinClick sadece drawer açıyor.
  3. Yakınlaşma, PinDetailDrawer.jsx içindeki butona basılınca tetikleniyor.
  4. Cluster tıklaması ise ayrı bir akışla doğrudan flyTo(...) çalıştırıyor.
Sonuç

Bu davranış bir veri çekme problemi değil. API bozuk olsaydı pin açılması, drawer içeriği veya koordinat bilgisi de tutarsız olurdu. Burada esas fark, tekil pin ve cluster için iki farklı UX kuralı yazılmış olması.

Mevcut Davranış Özeti

Tekil pin tıkla Yakınlaşmaz

Detay drawer açılır.

Cluster tıkla Yakınlaşır

Harita kümeye doğru zoom yapar.

Drawer butonu Yakınlaşır

Buraya yakınlaş ile fly-to çalışır.

API etkisi Düşük

Bu spesifik yakınlaşma şikayetinin ana nedeni değil.

Manual 2 Yorumu

Bu belge, mevcut sistemin gerçek davranışını yanlış beklenti oluşturmadan anlatmak için hazırlandı. Eğer önceki manual.html içinde “pine tıkla ve yakınlaş” gibi bir ifade varsa, o metin artık güncel kod davranışıyla uyuşmuyor olabilir.

Güncel kullanım dili böyle olmalı

  • “Cluster’a tıklarsan yakınlaşırsın.”
  • “Tek bir pine tıklarsan detay açılır.”
  • “Detay panelinden `Buraya yakınlaş` seçeneği ile o noktaya gidersin.”

İstersen Sonraki Kod Değişikliği

Eğer ürün beklentin gerçekten “tek pine tıklayınca hem drawer açılsın hem de kamera otomatik yakınlaşsın” ise, bu API değişikliği değil, frontend etkileşim değişikliğidir.

Gerekli frontend değişikliği

  • App.js içindeki onPinClick fonksiyonuna setFlyToCoords(...) eklenmeli.
  • Ardından drawer yine açılabilir.
  • Böylece tek tıkta hem odaklanma hem detay görünümü gelir.
Şu anki ürün kuralı

Tekil pin tıklaması ile cluster tıklaması aynı davranışı göstermiyor. Bu yüzden kullanıcının “yakınlaşması gerekirken saçmalıyor” hissi yaşaması anlaşılır.
